# Examen

L'évaluation de 60 minutes se portera sur les critères suivants :

- DevOps
  - [ ] Appliquer les principes DevOps.
  - [ ] Justifier l'utilisation de DevOps.
- Kubernetes
  - [ ] Corriger un déploiement Kubernetes.
  - [ ] Améliorer un déploiement Kubernetes.
- Observabilité
  - [ ] Rendre une application observable.
  - [ ] Proposer des technologies pertinentes.
  - [ ] Justifier l'observabilité.
- Concept
  - [ ] Expliquer des concepts liés à DevOps.
  - [ ] Comparer des concepts avec DevOps.
